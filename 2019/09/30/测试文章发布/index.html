学习一些感兴趣的内容

2019-09-30

昨天折腾了小半天，终于把hexo部署到了GitHub上，以后就在这里记录学习过程

目前希望能够掌握的有

  • python （web方向、数据分析）
  • Flash
  • Mongodb
  • Git
  • Docker
  • VS code
  • markdown
  • 树莓派

学习目的

  1. 从07年开始接触html和css，就感受编程的神奇和创造的乐趣，后来也接触了java，但由于没有长性，所以一直在从入门到放弃的阶段循环，现在毕业已经十年，有时候不经意看到某种编程语言实现的功能，还是会吊起心底那份兴趣，就像有的人喜欢书法、有的人喜欢钓鱼，我想把编程当成是一个业务爱好，不指望能有多厉害，能享受到这份乐趣足矣。
  1. 现在不少小学都已经加入了以python为主的编程教材，相信10出生的孩子，他们未来的世界，变成就跟开车一样，会成为非常普遍的一个技能，刚好我家里也有个小男孩，为了以后能和儿子多一点共同语言，就必须得开始学习python，过阵子再入手个树莓派，让孩子能实际感受到编程的效果，说不能还能增加他对我的崇拜，哈哈

文章添加图片

  1. 把本地Blog文件下的配置文件（非主题下）_config.yml里的post_asset_folder:选项设置为true
  2. 在本地Blog路径下执行npm install hexo-asset-image --save （用于下载安装一个可以上传本地图片的插件）
  3. 运行hexo n "xxxx"来生成md博文, 会发现在/source/_posts文件夹中除了xxxx.md`文件还生成了一个同名文件夹
  4. 最后在xxxx.md引入图片时，先把图片复制到xxxx文件夹中 然后在xxxx.md中按照markdown的格式引入图片：

![你想输入的替代文字](xxxx/图片名.jpg)
